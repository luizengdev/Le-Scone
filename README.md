## Projeto CSS SASS

Projeto simples de um menu de restaurante, para tornan a codificação do CSS mais eficiente e divertida. Projeto desenvolvido no curso da [Origamid](https://www.origamid.com/).



### Conceitos abordados
- Criar variáveis
- Mixins
- Funções
- Loops e outras funcionalidades
